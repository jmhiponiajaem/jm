My Learning Journey

  • HTML and CSS

    HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are fundamental technologies for building and designing web pages. HTML provides the structure and content of a web page, while CSS is used to control the presentation and layout.

  • JavaScript

    JavaScript is a versatile programming language commonly used for adding interactivity and dynamic features to websites. It runs on the client-side and can be used for tasks like form validation, DOM manipulation, and creating interactive web applications.

  • Python

    Python is a popular and versatile programming language known for its readability and simplicity. It can be used for various purposes, such as web development, data analysis, artificial intelligence, and scripting. Python's extensive library ecosystem makes it a powerful tool for different domains.

  • Java

    Java is a widely used, object-oriented programming language known for its platform independence. It is commonly used for developing desktop applications, mobile apps (Android), enterprise-level systems, and large-scale projects. Java's strong ecosystem and performance make it a popular choice in the industry.

  • SQL

    SQL (Structured Query Language) is a language used for managing and manipulating relational databases. It is essential for working with databases and performing tasks like storing, retrieving, and modifying data. SQL is a crucial skill for backend web development and data-driven applications.
